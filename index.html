音声翻訳

下のボタンを押している間、音声を取り込みます。
離すと翻訳が始まります。

⚙ 設定

翻訳モデル

💡 Claude 系 (Anthropic) と Gemini 系 (Google) でそれぞれ別の API キーが必要です。

Anthropic API Key

▶ APIキーの取得方法を表示
  1. Anthropic Console を開きアカウント作成 / ログイン
    console.anthropic.com を開く
  2. 左メニュー「API Keys」を選択
  3. 「Create Key」を押し、任意の名前を付けて作成
  4. 表示された sk-ant-... のキーをコピー
    ⚠️ キーは作成時の 1 回しか表示されません。コピー忘れに注意。
  5. 上の入力欄に貼り付けて [疎通テスト] → [保存]
💡 料金: Sonnet 4.6 は無料枠なし（従量課金）。100 翻訳あたり約 $0.18 (≒ 27 円) 程度。
⚠️ キーはこのブラウザの localStorage に保存されます。共用 PC では使用後に [キー削除]。

Google AI API Key (Gemini)

▶ APIキーの取得方法を表示
  1. Google アカウントでログイン
  2. Google AI Studio の API キー画面を開く
    aistudio.google.com/apikey を開く
  3. 「Create API key」を押し、プロジェクトを選択 (新規作成も可)
  4. 表示された AIza... のキーをコピー
  5. 上の入力欄に貼り付けて [疎通テスト] → [保存]
💡 無料枠: gemini-2.5-flash は 1日 数千リクエスト程度まで無料 (変動あり)。
⚠️ キーはこのブラウザの localStorage に保存されます。共用 PC では使用後に [キー削除]。

校正モード (録音音声で STT 結果を補正)

端末の音声認識は同音異義語に弱いため (例: 過酢酸→浅草)、録音した音声を AI に渡して校正します。 校正には音声を入力できる Gemini モデルが必要です。Google AI API キーが必須。

💡 ヒントは校正と翻訳の両方のプロンプトに含まれます。専門用語を列挙するほど精度が上がります。

デフォルト

マイク (音声入力)

▶ マイクが許可されない (not-allowed) 場合の対処

Android Chrome:

  1. アドレスバー左の 🔒 をタップ → 「権限」 → 「マイク」 → 「許可」
  2. または Android 設定 → アプリ → Chrome → 権限 → マイク → 「許可」
  3. Android 設定 → アプリ → 「Speech Services by Google」が有効か確認 (無効だと STT が動きません)
  4. Chrome を再起動 → このページを再読込 → [マイクをテスト]

iOS Safari:

  1. アドレスバー「ぁA」→ Web サイトの設定 → マイク → 許可
  2. iOS 設定 → Safari → マイク → 「確認」または「許可」
  3. iOS 設定 → プライバシーとセキュリティ → マイク → Safari がオン

PC (Chrome/Edge):

  1. アドレスバー左のアイコン → サイトの設定 → マイク → 許可
  2. OS のマイク権限 (Windows: 設定 → プライバシー → マイク) でブラウザが許可されているか
⚠️ 一度「ブロック」を選択すると、以降は許可ダイアログが出ずに即 not-allowed になります。 上記手順でサイトの権限を「リセット」または「許可」に手動変更する必要があります。

TTS 音声 (読み上げ)

▶ タイ語の音声が無い / 動かない場合の対処
⚠️ Web ブラウザの TTS は OS にインストールされている音声合成エンジンを使います。 Windows / macOS は標準ではタイ語の音声が入っていないことが多いため、追加が必要です。

Windows 10 / 11:

  1. 設定 → 時刻と言語 → 言語と地域
  2. 「言語を追加」→「ไทย / Thai」を追加
  3. 追加した言語を選択 → 「オプション」→ 「音声」をダウンロード
  4. 設定 → 時刻と言語 → 音声 で「Microsoft Achara / Pratip」等が表示されることを確認
  5. このページで [利用可能な音声を再読込] → 上の「タイ語」プルダウンに voice が表示されたら選択 → [保存]

macOS:

  1. システム設定 → アクセシビリティ → 読み上げコンテンツ → システムの声 → 「カスタマイズ…」
  2. 「Kanya (タイ語)」等にチェックを入れて DL
  3. このページで [利用可能な音声を再読込]

Chrome / Edge:

ブラウザ内蔵のオンライン音声 (Google / Microsoft) もタイ語をサポートします。インターネット接続必須。
企業ネットワーク等で *.google.com へのアクセスが制限されていると鳴らない場合があります。

💡 OS 追加が手間な場合、後日 Google Cloud TTS / Azure TTS のクラウド音声 (有料) に切り替える追加対応も可能です。
